Which panel features do you want?

Select all that should be available in the multi-function sidebar. "Files" and "Source Control" already exist — pick which new ones to add.

🗂

Vertical Tabs

Lists all open tabs as a scrollable sidebar column — title, type icon, and close button. Click to switch; replaces the top tab bar or supplements it.

🔍

Workspace Search

Find in files (grep across the whole project) — query box, file matches, line previews, click to open. Like VS Code's Search panel.

📋

Outline / Symbols

Shows the symbol tree of the active editor file (functions, classes, headings for Markdown). Click a symbol to jump to it.

🕓

Recently Opened Files

A quick-access list of recently visited files sorted by recency. Click to reopen without navigating the tree.
