## Step 1 — Get proot binary and Alpine rootfs

You need two pre-compiled assets. Do this **once** on your dev machine before anything else.

### 1a. Compile proot for Android ARM64

```bash
# On Linux/macOS. Requires Android NDK.
export NDK=$ANDROID_HOME/ndk/28.0.12916984   # adjust to your NDK version
export TOOLCHAIN=$NDK/toolchains/llvm/prebuilt/linux-x86_64/bin

git clone https://github.com/proot-me/proot.git
cd proot/src

# Cross-compile for aarch64-linux-android
make CC=$TOOLCHAIN/aarch64-linux-android33-clang \
     AR=$TOOLCHAIN/llvm-ar \
     STRIP=$TOOLCHAIN/llvm-strip \
     LOADER_NAMES="ld-musl-aarch64.so.1 ld-linux-aarch64.so.1" \
     proot

$TOOLCHAIN/llvm-strip proot
cp proot ../../src-tauri/assets/android/proot-aarch64
```

### 1b. Download Alpine Linux minimal rootfs

```bash
mkdir -p src-tauri/assets/android

# Download Alpine 3.19 aarch64 minimal rootfs (~8MB compressed)
curl -L "https://dl-cdn.alpinelinux.org/alpine/v3.19/releases/aarch64/alpine-minirootfs-3.19.1-aarch64.tar.gz" \
     -o src-tauri/assets/android/alpine-rootfs.tar.gz

# Verify checksum (get from Alpine download page)
sha256sum src-tauri/assets/android/alpine-rootfs.tar.gz
```

## Step 5 — Create `src-tauri/src/modules/pty/android.rs`

This is the new Android implementation using `nix::pty::openpty` + proot.

```rust
// src-tauri/src/modules/pty/android.rs
//
// Android PTY backend: opens /dev/ptmx via nix, spawns proot with
// an Alpine Linux rootfs, streams I/O back through a Tauri Channel.

use std::{
    collections::HashMap,
    fs,
    io::{Read, Write},
    os::unix::io::{FromRawFd, IntoRawFd, RawFd},
    path::PathBuf,
    process::{Child, Command, Stdio},
    sync::{Arc, Mutex, RwLock},
    thread,
};

use nix::{
    pty::{openpty, OpenptyResult, Winsize},
    unistd::setsid,
};
use serde::{Deserialize, Serialize};
use tauri::{AppHandle, Channel, Manager, Runtime, State};
use uuid::Uuid;

// ── Types ────────────────────────────────────────────────────────────────────

#[derive(Debug, Clone, Serialize, Deserialize)]
#[serde(rename_all = "camelCase")]
pub enum PtyEvent {
    Data { data: Vec<u8> },
    Exit { code: i32 },
}

struct Session {
    /// fd of the master side of the PTY
    master_fd: RawFd,
    /// handle to the proot child process
    child: Child,
    /// writer thread can write to the master fd via this sender
    write_tx: std::sync::mpsc::SyncSender<Vec<u8>>,
}

pub struct PtyState {
    sessions: RwLock<HashMap<String, Arc<Mutex<Session>>>>,
}

impl PtyState {
    pub fn new() -> Self {
        Self {
            sessions: RwLock::new(HashMap::new()),
        }
    }
}

// ── Helpers ───────────────────────────────────────────────────────────────────

/// Returns the path where the rootfs was extracted on first boot.
/// Typically: /data/data/app.crynta.terax/files/rootfs
fn rootfs_path(app: &AppHandle<impl Runtime>) -> PathBuf {
    app.path().app_data_dir()
        .expect("app_data_dir unavailable")
        .join("rootfs")
}

/// Returns the path of the proot binary copied on first boot.
fn proot_path(app: &AppHandle<impl Runtime>) -> PathBuf {
    app.path().app_data_dir()
        .expect("app_data_dir unavailable")
        .join("proot")
}

// ── Commands ──────────────────────────────────────────────────────────────────

/// Open a new PTY session backed by proot + Alpine.
/// Called from the frontend exactly like the desktop version.
#[tauri::command]
pub async fn pty_open<R: Runtime>(
    app: AppHandle<R>,
    state: State<'_, PtyState>,
    channel: Channel<PtyEvent>,
    cols: u16,
    rows: u16,
    cwd: Option<String>,
) -> Result<String, String> {
    let id = Uuid::new_v4().to_string();

    let rootfs = rootfs_path(&app);
    let proot  = proot_path(&app);

    // Sanity check — bootstrap should have run already
    if !rootfs.exists() || !proot.exists() {
        return Err("Bootstrap not complete. Call bootstrap_android first.".into());
    }

    // Open a PTY pair
    let winsize = Winsize {
        ws_col: cols,
        ws_row: rows,
        ws_xpixel: 0,
        ws_ypixel: 0,
    };
    let OpenptyResult { master, slave } =
        openpty(Some(&winsize), None).map_err(|e| e.to_string())?;

    let master_fd: RawFd = master.into_raw_fd();
    let slave_fd:  RawFd = slave.into_raw_fd();

    // Build the cwd inside the rootfs.
    // If the caller passed a host path we can't use it directly inside proot,
    // so we default to /root inside the Alpine rootfs.
    let inner_cwd = "/root";

    // Spawn proot
    //   -0           → fake root (uid 0), needed by Alpine's package manager
    //   -r <rootfs>  → rootfs directory
    //   -b /dev      → bind mount real /dev so the PTY slave is accessible
    //   -b /proc     → bind mount /proc for ps, top, etc.
    //   -b /sys      → bind mount /sys
    //   -w <cwd>     → set working directory inside the rootfs
    let child = unsafe {
        Command::new(proot.to_str().unwrap())
            .args([
                "-0",
                "-r", rootfs.to_str().unwrap(),
                "-b", "/dev",
                "-b", "/proc",
                "-b", "/sys",
                "-b", "/data:/data",   // expose host /data inside proot
                "-w", inner_cwd,
                // shell to launch
                "/bin/sh",
                "-l",                   // login shell — sources /etc/profile
            ])
            // Redirect all three std streams to the PTY slave
            .stdin(Stdio::from_raw_fd(slave_fd))
            .stdout(Stdio::from_raw_fd(slave_fd))
            .stderr(Stdio::from_raw_fd(slave_fd))
            // Set TERM so ncurses apps work
            .env("TERM", "xterm-256color")
            .env("COLORTERM", "truecolor")
            .env("HOME", "/root")
            .env("USER", "root")
            .env("PATH", "/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin")
            .spawn()
    }
    .map_err(|e| format!("Failed to spawn proot: {e}"))?;

    // Close the slave fd in the parent — the child owns it now
    unsafe { nix::libc::close(slave_fd) };

    // ── Read thread: master_fd → Channel ────────────────────────────────────
    let channel_read = channel.clone();
    let master_read  = master_fd;
    thread::spawn(move || {
        let mut buf = [0u8; 4096];
        let mut file = unsafe { fs::File::from_raw_fd(master_read) };
        loop {
            match file.read(&mut buf) {
                Ok(0) | Err(_) => {
                    let _ = channel_read.send(PtyEvent::Exit { code: 0 });
                    break;
                }
                Ok(n) => {
                    let _ = channel_read.send(PtyEvent::Data {
                        data: buf[..n].to_vec(),
                    });
                }
            }
        }
    });

    // ── Write channel: mpsc → master_fd ─────────────────────────────────────
    let (write_tx, write_rx) = std::sync::mpsc::sync_channel::<Vec<u8>>(64);
    let master_write = master_fd;
    thread::spawn(move || {
        let mut file = unsafe { fs::File::from_raw_fd(master_write) };
        for data in write_rx {
            if file.write_all(&data).is_err() {
                break;
            }
        }
    });

    let session = Session {
        master_fd,
        child,
        write_tx,
    };

    state
        .sessions
        .write()
        .unwrap()
        .insert(id.clone(), Arc::new(Mutex::new(session)));

    Ok(id)
}

/// Write bytes to the PTY (keyboard input from xterm.js).
#[tauri::command]
pub async fn pty_write(
    state: State<'_, PtyState>,
    id: String,
    data: Vec<u8>,
) -> Result<(), String> {
    let sessions = state.sessions.read().unwrap();
    let session  = sessions.get(&id).ok_or("Session not found")?;
    session
        .lock()
        .unwrap()
        .write_tx
        .send(data)
        .map_err(|e| e.to_string())
}

/// Resize the PTY window (called when the xterm.js container resizes).
#[tauri::command]
pub async fn pty_resize(
    state: State<'_, PtyState>,
    id: String,
    cols: u16,
    rows: u16,
) -> Result<(), String> {
    let sessions = state.sessions.read().unwrap();
    let session  = sessions.get(&id).ok_or("Session not found")?;
    let master_fd = session.lock().unwrap().master_fd;

    let ws = Winsize {
        ws_col: cols,
        ws_row: rows,
        ws_xpixel: 0,
        ws_ypixel: 0,
    };

    // TIOCSWINSZ ioctl
    unsafe {
        nix::libc::ioctl(master_fd, nix::libc::TIOCSWINSZ, &ws);
    }

    Ok(())
}

/// Close a PTY session.
#[tauri::command]
pub async fn pty_close(
    state: State<'_, PtyState>,
    id: String,
) -> Result<(), String> {
    let mut sessions = state.sessions.write().unwrap();
    if let Some(session) = sessions.remove(&id) {
        let mut s = session.lock().unwrap();
        let _ = s.child.kill();
        unsafe { nix::libc::close(s.master_fd) };
    }
    Ok(())
}
```

## Step 10 — Create `src-tauri/src/bootstrap.rs`

This Rust command is called **once** from the frontend on first launch. It extracts proot and the Alpine rootfs from the app's asset bundle to internal storage.

```rust
// src-tauri/src/bootstrap.rs
// Android-only. Extracts proot binary + Alpine rootfs to app internal storage.

use std::{
    fs,
    io::Write,
    os::unix::fs::PermissionsExt,
    path::PathBuf,
};
use tauri::{AppHandle, Manager, Runtime};

fn data_dir(app: &AppHandle<impl Runtime>) -> PathBuf {
    app.path().app_data_dir().expect("no app_data_dir")
}

#[tauri::command]
pub async fn bootstrap_android<R: Runtime>(
    app: AppHandle<R>,
) -> Result<String, String> {
    let base    = data_dir(&app);
    let proot   = base.join("proot");
    let rootfs  = base.join("rootfs");

    // Already bootstrapped?
    if proot.exists() && rootfs.exists() {
        return Ok("already_bootstrapped".into());
    }

    fs::create_dir_all(&base).map_err(|e| e.to_string())?;

    // ── 1. Copy proot binary ────────────────────────────────────────────────
    // Tauri bundles files in `assets/` and exposes them via `resolve_resource`.
    let proot_src = app
        .path()
        .resolve("assets/android/proot-aarch64", tauri::path::BaseDirectory::Resource)
        .map_err(|e| e.to_string())?;

    fs::copy(&proot_src, &proot).map_err(|e| e.to_string())?;

    // Make it executable
    fs::set_permissions(&proot, fs::Permissions::from_mode(0o755))
        .map_err(|e| e.to_string())?;

    // ── 2. Extract Alpine rootfs ────────────────────────────────────────────
    let rootfs_src = app
        .path()
        .resolve("assets/android/alpine-rootfs.tar.gz", tauri::path::BaseDirectory::Resource)
        .map_err(|e| e.to_string())?;

    fs::create_dir_all(&rootfs).map_err(|e| e.to_string())?;

    // Use the `tar` command bundled with Android (available since Android 6)
    let status = std::process::Command::new("tar")
        .args([
            "xzf",
            rootfs_src.to_str().unwrap(),
            "-C",
            rootfs.to_str().unwrap(),
        ])
        .status()
        .map_err(|e| e.to_string())?;

    if !status.success() {
        return Err("tar extraction failed".into());
    }

    // ── 3. Write a basic /etc/resolv.conf for DNS ──────────────────────────
    let resolv = rootfs.join("etc/resolv.conf");
    let mut f = fs::File::create(resolv).map_err(|e| e.to_string())?;
    writeln!(f, "nameserver 8.8.8.8").map_err(|e| e.to_string())?;
    writeln!(f, "nameserver 1.1.1.1").map_err(|e| e.to_string())?;

    Ok("bootstrapped".into())
}
```

## Step 14 — Create `BootstrapPlugin.kt`

After `tauri android init`, this file goes here:
`src-tauri/gen/android/app/src/main/java/app/crynta/terax/BootstrapPlugin.kt`

```kotlin
package app.crynta.terax

import android.content.Context
import app.tauri.annotation.Command
import app.tauri.annotation.TauriPlugin
import app.tauri.plugin.Invoke
import app.tauri.plugin.Plugin
import java.io.File
import java.io.FileOutputStream

/**
 * BootstrapPlugin is not actually used at runtime —
 * the bootstrap is done from Rust (bootstrap.rs) using the `tar` binary.
 * This plugin exists as a fallback for older Android versions
 * where the system `tar` may not be available, using Java's ZipInputStream instead.
 */
@TauriPlugin
class BootstrapPlugin(private val context: Context) : Plugin(context) {

    @Command
    fun isBootstrapped(invoke: Invoke) {
        val base   = context.filesDir
        val proot  = File(base, "proot")
        val rootfs = File(base, "rootfs")
        invoke.resolve(mapOf("ready" to (proot.exists() && rootfs.exists())))
    }

    @Command
    fun extractRootfs(invoke: Invoke) {
        // Called only if Rust-side tar extraction failed (Android < 6).
        // Uses Java streams to unpack the gzipped tar from assets.
        try {
            val base   = context.filesDir
            val rootfs = File(base, "rootfs").also { it.mkdirs() }

            // Copy proot binary
            val proot = File(base, "proot")
            context.assets.open("android/proot-aarch64").use { input ->
                FileOutputStream(proot).use { output -> input.copyTo(output) }
            }
            proot.setExecutable(true, false)

            // Unpack rootfs tarball using Android's built-in tar
            val tarball = File(context.cacheDir, "alpine-rootfs.tar.gz")
            context.assets.open("android/alpine-rootfs.tar.gz").use { input ->
                FileOutputStream(tarball).use { output -> input.copyTo(output) }
            }

            val proc = ProcessBuilder("tar", "xzf", tarball.absolutePath, "-C", rootfs.absolutePath)
                .redirectErrorStream(true)
                .start()
            val exitCode = proc.waitFor()
            tarball.delete()

            if (exitCode != 0) {
                invoke.reject("tar extraction failed with code $exitCode")
            } else {
                // Write resolv.conf
                File(rootfs, "etc/resolv.conf").writeText("nameserver 8.8.8.8\nnameserver 1.1.1.1\n")
                invoke.resolve(mapOf("status" to "ok"))
            }
        } catch (e: Exception) {
            invoke.reject(e.message ?: "Unknown error")
        }
    }
}
```

## Step 15 — Create `KeystorePlugin.kt`

`src-tauri/gen/android/app/src/main/java/app/crynta/terax/KeystorePlugin.kt`

```kotlin
package app.crynta.terax

import android.content.Context
import android.security.keystore.KeyGenParameterSpec
import android.security.keystore.KeyProperties
import app.tauri.annotation.Command
import app.tauri.annotation.InvokeArg
import app.tauri.annotation.TauriPlugin
import app.tauri.plugin.Invoke
import app.tauri.plugin.Plugin
import java.security.KeyStore
import javax.crypto.Cipher
import javax.crypto.KeyGenerator
import javax.crypto.SecretKey
import javax.crypto.spec.GCMParameterSpec
import android.util.Base64
import java.io.File

/**
 * KeystorePlugin — replaces the `keyring` crate on Android.
 * Uses Android Keystore (hardware-backed on most devices) + AES-256-GCM.
 * Encrypted blobs are stored as files in app private storage, one per key.
 */
@InvokeArg
class KeyArgs {
    lateinit var service: String
    lateinit var account: String
    var secret: String? = null
}

@TauriPlugin
class KeystorePlugin(private val context: Context) : Plugin(context) {

    companion object {
        private const val KEYSTORE_PROVIDER = "AndroidKeyStore"
        private const val CIPHER_ALGO = "AES/GCM/NoPadding"
        private const val KEY_SIZE = 256
        private const val GCM_TAG_LENGTH = 128
    }

    // One AES key per (service, account) pair, stored in Android Keystore
    private fun keystoreAlias(service: String, account: String) =
        "terax:$service:$account"

    private fun storageFile(service: String, account: String): File {
        val dir = File(context.filesDir, "keystore_secrets").also { it.mkdirs() }
        // Sanitize to safe filename
        val name = Base64.encodeToString(
            "$service:$account".toByteArray(),
            Base64.URL_SAFE or Base64.NO_WRAP
        )
        return File(dir, name)
    }

    private fun getOrCreateKey(alias: String): SecretKey {
        val ks = KeyStore.getInstance(KEYSTORE_PROVIDER).also { it.load(null) }
        if (ks.containsAlias(alias)) {
            return (ks.getEntry(alias, null) as KeyStore.SecretKeyEntry).secretKey
        }
        val keyGen = KeyGenerator.getInstance(KeyProperties.KEY_ALGORITHM_AES, KEYSTORE_PROVIDER)
        keyGen.init(
            KeyGenParameterSpec.Builder(
                alias,
                KeyProperties.PURPOSE_ENCRYPT or KeyProperties.PURPOSE_DECRYPT
            )
                .setBlockModes(KeyProperties.BLOCK_MODE_GCM)
                .setEncryptionPaddings(KeyProperties.ENCRYPTION_PADDING_NONE)
                .setKeySize(KEY_SIZE)
                .setUserAuthenticationRequired(false)   // no biometric required
                .build()
        )
        return keyGen.generateKey()
    }

    @Command
    fun get(invoke: Invoke) {
        val args = invoke.parseArgs(KeyArgs::class.java)
        try {
            val file = storageFile(args.service, args.account)
            if (!file.exists()) {
                invoke.resolve(mapOf("value" to null))
                return
            }
            val blob  = file.readBytes()
            // First 12 bytes = IV, rest = ciphertext+tag
            val iv         = blob.copyOfRange(0, 12)
            val ciphertext = blob.copyOfRange(12, blob.size)
            val key    = getOrCreateKey(keystoreAlias(args.service, args.account))
            val cipher = Cipher.getInstance(CIPHER_ALGO)
            cipher.init(Cipher.DECRYPT_MODE, key, GCMParameterSpec(GCM_TAG_LENGTH, iv))
            val plain = String(cipher.doFinal(ciphertext))
            invoke.resolve(mapOf("value" to plain))
        } catch (e: Exception) {
            invoke.reject(e.message ?: "get failed")
        }
    }

    @Command
    fun set(invoke: Invoke) {
        val args = invoke.parseArgs(KeyArgs::class.java)
        val secret = args.secret ?: return invoke.reject("secret is required")
        try {
            val key    = getOrCreateKey(keystoreAlias(args.service, args.account))
            val cipher = Cipher.getInstance(CIPHER_ALGO)
            cipher.init(Cipher.ENCRYPT_MODE, key)
            val iv         = cipher.iv                    // 12 bytes, GCM auto-generated
            val ciphertext = cipher.doFinal(secret.toByteArray())
            val blob = iv + ciphertext                    // prepend IV to ciphertext
            storageFile(args.service, args.account).writeBytes(blob)
            invoke.resolve(mapOf("ok" to true))
        } catch (e: Exception) {
            invoke.reject(e.message ?: "set failed")
        }
    }

    @Command
    fun delete(invoke: Invoke) {
        val args = invoke.parseArgs(KeyArgs::class.java)
        try {
            val file = storageFile(args.service, args.account)
            if (file.exists()) file.delete()
            // Also remove the Keystore key so a future set() creates a fresh one
            val ks = KeyStore.getInstance(KEYSTORE_PROVIDER).also { it.load(null) }
            val alias = keystoreAlias(args.service, args.account)
            if (ks.containsAlias(alias)) ks.deleteEntry(alias)
            invoke.resolve(mapOf("ok" to true))
        } catch (e: Exception) {
            invoke.reject(e.message ?: "delete failed")
        }
    }
}
```

## Step 18 — Frontend: touch input for xterm.js

Create this file:

```typescript
// src/modules/terminal/touch.ts
// Adds mobile touch keyboard support to an xterm.js Terminal instance.
// On Android, tapping the terminal opens the soft keyboard via a hidden input.

import type { Terminal } from "@xterm/xterm";

export function installTouchInput(terminal: Terminal, container: HTMLElement) {
  // Create an invisible <input> that pulls up the soft keyboard on tap
  const hiddenInput = document.createElement("input");
  hiddenInput.setAttribute("autocorrect", "off");
  hiddenInput.setAttribute("autocapitalize", "none");
  hiddenInput.setAttribute("spellcheck", "false");
  hiddenInput.setAttribute("inputmode", "text");
  hiddenInput.style.cssText = `
    position: fixed;
    left: -9999px;
    top: 0;
    width: 1px;
    height: 1px;
    opacity: 0;
    border: none;
    outline: none;
  `;
  document.body.appendChild(hiddenInput);

  // Tap on terminal → focus hidden input → keyboard appears
  container.addEventListener("click", () => {
    hiddenInput.focus();
  });

  // Forward typed characters to the terminal
  hiddenInput.addEventListener("input", (e) => {
    const inputEl = e.target as HTMLInputElement;
    const text = inputEl.value;
    if (text) {
      terminal.paste(text);      // paste() sends text as if typed
      inputEl.value = "";        // clear so next input event fires cleanly
    }
  });

  // Forward special keys (backspace, enter, arrows)
  hiddenInput.addEventListener("keydown", (e) => {
    const keyMap: Record<string, string> = {
      Backspace: "\x7f",
      Enter:     "\r",
      ArrowUp:   "\x1b[A",
      ArrowDown: "\x1b[B",
      ArrowRight:"\x1b[C",
      ArrowLeft: "\x1b[D",
      Tab:       "\t",
      Escape:    "\x1b",
    };
    const seq = keyMap[e.key];
    if (seq) {
      e.preventDefault();
      terminal.paste(seq);
    }
  });

  // Return cleanup function
  return () => {
    document.body.removeChild(hiddenInput);
  };
}
```

## Troubleshooting

**`tar: not found` during bootstrap**
Android < 6 may lack `tar`. The `BootstrapPlugin.kt` has a Java fallback — call `bootstrap_extract_rootfs` from Kotlin instead of relying on the Rust `tar` call.

**PTY reads nothing**
Make sure the slave fd is closed in the parent after `spawn()`. The read thread blocks until the slave side has a writer (the child process). If slave fd is still open in the parent, closing master won't generate EOF.

**proot crashes with `SIGSEGV` on ARM**
This means the proot binary wasn't compiled for the correct ABI. Recompile with `aarch64-linux-android33-clang` and verify with `file proot-aarch64` — it should say `ELF 64-bit LSB shared object, ARM aarch64`.

**Soft keyboard doesn't appear**
The `hiddenInput.focus()` must be called from a user gesture handler (the `click` event). Android blocks `.focus()` calls that aren't in a touch/click handler.

**Keystore: `UserNotAuthenticated` exception**
This happens if you set `setUserAuthenticationRequired(true)`. The `KeystorePlugin.kt` above sets it to `false`. If you want biometric protection later, set it to `true` and handle the `BiometricPrompt` flow.
